# 飞书配置说明 · 家庭诊断问卷

> 问卷已改为**飞书多维表格 + 飞书群机器人**，不需要Google账号。
> 只需在 `index.html` 顶部 `FEISHU = {...}` 里填入4个值即可上线。

---

## 第一步：创建飞书多维表格（数据库）

1. 打开 [https://feishu.cn](https://feishu.cn) → 新建「多维表格」
2. 重命名表格，建议列名如下（**必须与 `index.html` 里的 `fields` 键名一致**）：

| 列名 | 类型 | 说明 |
|------|------|------|
| 提交时间 | 日期时间 | 自动记录 |
| 姓名 | 单行文本 | 填写人姓名 |
| 手机 | 电话 | 联系方式 |
| 微信号 | 单行文本 | 选填 |
| 关系 | 单行文本 | 与孩子的关系 |
| 城市 | 单行文本 | 所在城市 |
| 出国方式 | 单选 | 见问卷选项 |
| 目标国家 | 单行文本 | |
| 家庭综合得分 | 数字 | 0-100 |
| 家庭等级 | 单选 | 高度就绪/中度就绪/需关注/重点提升 |
| 孩子详情 | 多行文本 | 自动生成 |

3. **获取 `APP_TOKEN`**：打开多维表格，地址栏URL类似：
   ```
   https://xxxx.feishu.cn/bitable/MFyjRKxxxxxxxxxxxxxxx/
   ```
   `M` 开头那串就是 `APP_TOKEN`

4. **获取 `TABLE_ID`**：点进表格视图，URL变成：
   ```
   .../bitable/MFyjRKxx/tblSHMxxxxxxxxxxxx/
   ```
   `tbl` 开头那串就是 `TABLE_ID`

---

## 第二步：获取 tenant_access_token

1. 打开 [https://open.feishu.cn](https://open.feishu.cn) → 登录 → 「开发者后台」
2. 创建自建应用（随便起名，如"家庭诊断"）
3. 在「凭证与基础信息」里找到：
   - `App ID`
   - `App Secret`
4. 在应用管理页开启权限：`bitable:app`（读写多维表格）
5. 获取 token（后台执行一次，或配成环境变量）：
   ```bash
   curl -X POST 'https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal' \
     -H 'Content-Type: application/json' \
     -d '{
       "app_id": "YOUR_APP_ID",
       "app_secret": "YOUR_APP_SECRET"
     }'
   ```
   返回 JSON 里的 `tenant_access_token` 值填入 `ACCESS_TOKEN`

---

## 第三步：添加飞书群机器人（通知你）

1. 打开你的飞书群 → 右上角「···」→「群机器人」→「添加机器人」
2. 选择「自定义机器人」→ 随便起名 → 复制 **Webhook 地址**
3. 把 Webhook 地址填入 `WEBHOOK_URL`

---

## 第四步：填入 `index.html`

打开 `index.html`，找到第 964 行附近的：

```javascript
const FEISHU = {
  APP_TOKEN:    'YOUR_APP_TOKEN_HERE',
  TABLE_ID:     'YOUR_TABLE_ID_HERE',
  ACCESS_TOKEN: 'YOUR_TENANT_ACCESS_TOKEN_HERE',
  WEBHOOK_URL: 'YOUR_WEBHOOK_URL_HERE',
};
```

把四个值替换成上面获取到的真实值，保存即可。
